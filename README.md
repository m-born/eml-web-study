Data for the Economic Media Lab website.

# Filetype to folder structure

* ``attachments`` ← Images, documents, videos, …
* ``html`` ← HTML files
* ``md`` ← MD files
* …

# File naming and content conventions for new MD files

## File naming
* Use lowercase only
* Replace spaces with dashes

Like so:
```
my-new-file.md
```

## File content
* First line: ``# HEADING``
* Second line: empty
* Third line fwd: everything else

Like so:
```
# HEADING

EVERYTHING ELSE
```
